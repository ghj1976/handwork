[直流电机和交流电机的原理和区别是什么？](https://www.zhihu.com/question/24086069)

* 有刷电机  简单的说就是有碳刷的电机(对于航模来说,马达高转速下,碳刷比较容易磨损),有刷电机制造工艺简单,价格相对较低。

* 无刷电机  简单的说就是没有碳刷的电机 ,无刷电机 分为内转子和外转子两种,(无刷电机目前是模型的主流动力)
